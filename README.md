# simple
一个基于vue 的组件库

# 项目运行

```
# 克隆到本地
git clone https://github.com/simpleTeam3/simple.git

# 进入项目
cd simple

# 安装依赖
npm install，可以使用cnpm代替npm

#运行项目
npm run dev
```

# 项目目录结构

```
├── assets                      ---静态资源
├── examples                    ---用例展示
├── doc				                  ---相关开发文档
├── src				                  ---源码
  ├── common                    ---公共   
  ├── components		            ---组件
  ├── mixins                    ---混合方法
  ├── style			                ---样式
  ├── utils			                ---工具库
  └── index.js                  ---组件库入口文件
├── package.json                -- 项目配置和依赖
```
